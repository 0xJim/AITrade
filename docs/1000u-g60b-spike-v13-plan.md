# 1000U 模拟盘组合运行计划：G60B + Spike-v13-P4

## 目标

用 1000U 做一个进攻偏均衡的组合模拟盘：

- G60B 做主策略，负责稳定和低回撤。
- Spike-v13-P4 做小仓进攻，捕捉 15m 收线异动。
- 保留现金缓冲，防止连续亏损时被动。

本计划默认先跑模拟盘/测试网，不直接切实盘。

## 当前回测依据

### G60B

- 回测周期：2025-05-14 10:00 到 2026-05-14 10:00
- 初始资金：1000U
- 手续费：0.04%
- 滑点：0.05%
- 结果：+601.27U
- 最大回撤：3.31%
- 定位：防守型主策略

### Spike-v13-P4

- 回测周期：2025-05-14 10:00 到 2026-05-14 10:00
- 初始资金：1000U
- 手续费：0.04%
- 滑点：0.05%
- 结果：+1060.85U
- 最大回撤：6.59%
- 定位：小仓进攻策略

原始 Spike v13 严格版虽然收益更高，约 +2652U，但最大回撤 14.36%，不适合作为 1000U 小账户主策略。

## 执行方案：直接进入正式模拟盘组合

| 模块 | 资金 | 占比 | 策略 | 用途 |
|---|---:|---:|---|---|
| 主仓 | 600U | 60% | G60B | 主策略、防守、稳定 |
| 进攻仓 | 300U | 30% | Spike-v13-P4 | 进攻模块、捕捉 15m 异动 |
| 现金 | 100U | 10% | 不交易 | 安全垫 |

### G60B 参数

- 策略：G60B
- 资金上限：600U
- 最大同时持仓：3
- 单笔保证金参考：约 60U
- 15m 异动候选池：开启
- 15m 异动阈值：1.0%
- 只跑测试网/模拟盘

### Spike-v13-P4 参数

- 策略：Spike-v13-P4
- 资金上限：300U
- 单笔保证金：约 4% 总账户，即 40U 以内
- 高质量信号加仓后仍不超过 52U
- 最大同时持仓：最多 3 笔
- 信号来源：15m 已收线涨幅 >= 1%
- 方向：只做多
- 过滤：1h EMA 多头、RSI >= 50、质量分 >= 70
- 最大持仓：8 小时

## 风控暂停规则

这些规则必须硬执行，不做主观加仓。

### 总账户规则

- 账户权益 <= 950U：暂停 Spike-v13-P4，只保留 G60B。
- 账户权益 <= 900U：全部暂停，复盘后再恢复。
- 任意一天总亏损 >= 50U：当天停止新开仓。
- 禁止补仓摊平。
- 禁止手动加杠杆。

### Spike-v13-P4 规则

- 单日亏损 >= 25U：暂停当天。
- 连续亏损 >= 3 笔：暂停 24 小时。
- 单策略回撤 >= 8%：暂停，复盘。
- 最大同时持仓：最多 3 笔。
- 分配资金可以是 300U，但单笔保证金不能跟着放大，仍按 40U 左右执行。
- 不允许超过分配资金上限。

### G60B 规则

- 连续亏损 >= 5 笔：暂停 24 小时。
- 单策略回撤 >= 5%：暂停，复盘。
- API 未连通时只监控，不下单。
- 没有交易所级止损时，不允许实盘。

## 每日检查项

每天至少检查一次：

- 总账户权益。
- 当前持仓数量。
- 今日已实现盈亏。
- Spike 是否触发暂停规则。
- G60B 是否触发暂停规则。
- 是否有 API 未连通、下单失败、止损挂单失败。
- `trades.json` 是否正常记录。
- 是否有持仓但交易所无对应仓位。

## 结论

1000U 账户不适合全仓跑 Spike v13。推荐路径是：

1. 直接执行：600U G60B + 300U Spike-v13-P4 + 100U 现金。
2. 任何时候 DD 或连续亏损触发规则，都先暂停 Spike，再判断是否暂停全部。
3. 不加仓摊平，不临时扩大 Spike 仓位。

G60B 是底仓，Spike-v13-P4 是小仓进攻模块。
