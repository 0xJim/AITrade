# AITrade - 加密货币交易系统 v11j

自动化加密货币合约交易系统，基于六维评分框架 + Kelly动态仓位管理。

## 架构概述

```
trading/
├── cron_scan.py          # 调度入口（v8评分框架）
├── config.py             # 策略配置
├── backtest.py           # 主力回测引擎（~1300行, 拉Binance正式API）
├── backtest_1year.py     # 1年期回测专用版
├── backtest_*.py         # 各版本回测脚本
├── binance_api.py        # Binance API封装（多时间框架）
├── review_db.py          # 复盘SQLite数据库
├── notifier.py           # 通知模块（Telegram + 微信）
├── analyze_*.py          # 数据分析/可视化脚本
└── archive/              # 历史版本归档
```

## v11j 核心逻辑

- **信号评分**：六维加权（OI 20% + 费率 15% + 量价 25% + 宏观 15% + 清算 10% + 聪明钱 15%）
- **信号质量**：量价40+形态30+订单流20+宏观10，≥65分
- **入场三要素**：信号≥65 + RR≥1.5 + 宏观30-70
- **仓位管理**：Kelly动态仓位（2-20%）
- **风控**：单笔亏损上限$40，硬止损
