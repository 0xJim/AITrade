## ⚙️ 核心系统组件

## 📜 版本说明

- 所有回测使用币安正式API(fapi.binance.com)历史数据
- v11j回测时间跨度：1000天(约2.7年)
- 原始本金有5000U和1000U两种，已统一换算为1000U等效值
- 5000U→1000U换算方法：PnL × (1000/初始本金)，回撤百分比不变
- v11g/v11h/v11i/v11j是离线模拟（基于v10原始数据+仓位调整），非实时拉数据
- **S22 JSON结果文件仍是旧数据(硬截断)**，需要用修正后的脚本重跑生成
