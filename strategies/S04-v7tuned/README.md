## 🧠 策略逻辑

## v7tuned — 精准狙击版（收紧参数）
**策略核心：** 与v7plus相同的策略框架但参数更保守。ATR动态止损×3.0，低ATR宽止损防止被扫。启用连亏趋势反转确认（REVERSAL_CONFIRM_ENABLED=True），连亏后需5根反转K线才能重新入场。环境评分MIN_ENV_SCORE=5（更严格）。MAX_SL_PCT=8%（v7plus为15%），控制高ATR币种单笔亏损。趋势跟随触发3%（比v7plus的1.5%更晚）。分批止盈50%+保本。
**出场：** SL止损、分批止盈+趋势跟随、移动止盈、时间止损48h。
**仓位：** 固定RISK_PER_TRADE=1%，杠杆3倍，最多2仓位。
**与上一版差异：** v7+→v7tuned：风险降至1%，仓位降至2，ENV门槛升至5，MAX_SL收紧到8%，启用反转确认，趋势跟随触发从1.5%升至3%，时间止损从72h降至48h，连亏冷却7天。
