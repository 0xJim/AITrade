## 🧠 策略逻辑

## v7plus — 精准狙击版（开放参数）
**策略核心：** 基于v6数据优化的宽松版本。入场依赖极端费率信号（负费率做多/正费率做空）+ 环境评分（MIN_ENV_SCORE=3，较宽松）。ATR动态止损：低ATR币种SL=ATR×3.0，防止窄止损被扫。分批止盈50%后剩余仓位用EMA9趋势跟随，不设TP上限让利润奔跑。趋势跟随触发1.5%，移动止盈5%触发/4%回撤锁定。时间止损72h（延长自48h），36h开始衰减。
**出场：** SL止损（ATR动态）、分批止盈+趋势跟随、移动止盈、时间止损。
**仓位：** 固定RISK_PER_TRADE=1.2%，杠杆3倍，最多3仓位。
**与上一版差异：** v6→v7+：风险从1%提至1.2%，仓位从2增到3，冷却从24h降至12h，连亏冷却从7天降至4天，MIN_ENV_SCORE从4降至3，极端费率阈值放宽（-0.05/-0.08），关闭反转确认。
