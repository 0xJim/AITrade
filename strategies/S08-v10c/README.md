## v10c — 数据驱动优化版（当前主版本backtest.py）
**策略核心：** 基于v10b（39笔/+$958/19.2%）数据分析优化。核心发现：SL 3-4%区间90%胜率、做空71%胜率优于做多45%、持仓<12h全被SL扫亏$246、持仓>48h胜率88%。三大优化：(1)入场后4h宽限期（GRACE_PERIOD_HOURS=4），不检查SL；(2)MAX_HOLD延长到72h（原48h）；(3)初始资金$1000。ATR动态止损双档：低ATR×3.0/高ATR×1.5。MAX_SL_PCT=8%。做空env门槛+2。动态仓位：env≥7加成×1.2，仓位上限20%，每笔最大亏损2%。连亏反转确认+趋势跟随5%触发+分批止盈50%+保本+移动止盈5%/4%。
**出场：** ATR动态SL（含4h宽限期）、分批止盈+趋势跟随、移动止盈、时间止损72h。
**仓位：** 固定1.5%风险，env≥7加成×1.2，杠杆3倍，最多3仓位。
**与上一版差异：** v10→v10c：新增4h宽限期，MAX_HOLD 48→72h，风险2%→1.5%，仓位2→3，初始资金改为$1000，MAX_SL_PCT从5%→8%，env做空门槛调整。
