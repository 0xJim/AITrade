## ⚠️ Bug修正记录 (2026-05-09)

**离线模拟器发现重大bug：** `backtest_v11j_compare.py` 的 `simulate()` 函数使用"亏损硬截断"逻辑：
```python
# 旧逻辑（错误）：只截亏损，盈利不变 → 虚高收益
if pnl < 0 and abs(pnl) > max_loss_per_trade:
    pnl = -max_loss_per_trade
```

**修正为"开仓前缩仓"：** 估算亏损超限时，按比例缩小仓位，盈亏同比缩小：
```python
# 新逻辑（正确）：盈亏同比缩放
if raw_pnl < 0 and abs(raw_pnl) * mult > max_loss_per_trade:
    shrink_ratio = max_loss_per_trade / (abs(raw_pnl) * mult)
    mult *= shrink_ratio
pnl = raw_pnl * mult
```
