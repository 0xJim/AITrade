## v10 — 三大升级版（15mK线+双向+费率异动）
**策略核心：** 三大升级：(1)真实15分钟K线数据，不再用1h近似；(2)完整双向交易，涨了做多+跌了做空；(3)费率异动作为独立触发条件。触发条件满足任一即可：A)单根15m K线涨跌幅>阈值(SPIKE_THRESHOLD=1%)；B)费率超出极端阈值（做多费率<-5%/做空费率>5%）。入场过滤：六维评分+多时间框架一致性+信号质量≥55+RR≥1.3+方向×质量×ATR过滤（做多ATR<5%+quality≥80，做空quality≥85+ATR≥3%）。Kelly仓位，10个时间框架分析。
**出场：** ATR动态SL（×1.5）、分批止盈50%+保本、趋势跟随、移动止盈。
**仓位：** Kelly动态（保守系数0.25），杠杆3倍，最多2仓位。
**与上一版差异：** v8→v10：新增15m真实K线、完整做空逻辑、费率异动独立触发、异动扫描器（1%阈值）、10个MTF框架。参数继续从yaml加载。
